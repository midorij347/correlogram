コレログラム解析レポート

RStudio を入れられない環境向けに、RStudio っぽく時系列解析を進めるための軽量HTML を作成しました。 このページ単体で、コレログラムの意味・見方・実装のたたき台を確認できます。

目的

時系列データに対して自己相関を調べ、周期性や依存構造の候補を見つけることです。

1. サンプル時系列

24周期と8周期を混ぜ、ノイズを加えた例です。

2. コレログラム

ラグごとの自己相関を可視化しています。

読み方

  • ラグ0の自己相関は必ず1です。
  • 同じような山が一定間隔で現れるなら、周期性の候補があります。
  • この例ではラグ24付近に強い構造が出るため、24周期の存在を疑えます。

Python 実装

import numpy as np
import matplotlib.pyplot as plt

def acf_values(x, nlags=60):
    x = np.asarray(x, dtype=float)
    x = x - x.mean()
    denom = np.dot(x, x)
    vals = [1.0]
    for lag in range(1, nlags + 1):
        vals.append(np.dot(x[:-lag], x[lag:]) / denom)
    return np.array(vals)

acf = acf_values(y, nlags=60)

plt.figure(figsize=(10, 4))
plt.stem(range(len(acf)), acf)
plt.axhline(0, linewidth=1)
plt.title("Correlogram")
plt.xlabel("Lag")
plt.ylabel("ACF")
plt.show()

R で書くと

x <- ts(your_vector)
acf(x, lag.max = 60, main = "Correlogram")

自分のデータに差し替えるとき

CSVの1列を value として読み込み、同じ関数に渡せばそのまま解析できます。

import pandas as pd
mydf = pd.read_csv("your_timeseries.csv")
x = mydf["value"].to_numpy()
acf = acf_values(x, nlags=60)
