コレログラム解析レポート（R版）

RStudio を入れられない環境向けに、R 言語のコードでそのまま再現できる 形のレポートです。

このHTMLでわかること

  • R でサンプル時系列を作る方法
  • plot() と acf() で時系列とコレログラムを描く方法
  • コレログラムの読み方
  • 自分のCSVに差し替える最小テンプレート

Rコード

set.seed(7)

n <- 240
time <- 1:n

value <- 0.03 * time +
  1.4 * sin(2 * pi * time / 24) +
  0.6 * sin(2 * pi * time / 8) +
  rnorm(n, mean = 0, sd = 0.5)

df <- data.frame(time = time, value = value)

plot(
  df$time, df$value, type = "l",
  main = "Sample Time Series",
  xlab = "Time",
  ylab = "Value"
)

acf(
  df$value,
  lag.max = 60,
  main = "Correlogram (Autocorrelation)"
)

読み方

  • ラグ 0 の自己相関は 1 です。
  • ラグ 24 付近など、一定間隔でピークが出るなら周期性の候補です。
  • この例では 24 周期と 8 周期を混ぜているため、その影響が自己相関にも現れます。

自分のCSVに差し替える

mydf <- read.csv("your_timeseries.csv")
acf(mydf$value, lag.max = 60, main = "Your Correlogram")
